Settings
========


``URCHIN_ID``
-------------

The identifier assigned to your `Google Analytics`_ account.  If this is not
found in the ``settings.py`` file, an empty string is returned instead of the
javascript block.

::

    URCHIN_ID = 'XXXXXXXXX'
